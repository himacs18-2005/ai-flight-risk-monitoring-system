# AI-Based Flight Risk Monitoring System

## Features
- Real-time weather using OpenWeather API
- AI-based risk prediction (Low / Medium / High)
- Stores flight data with timestamp
- Auto mode simulation
- Chart visualization using Chart.js

## Tech Stack
- Java (Spring Boot)
- HTML, CSS, JavaScript
- H2 Database
- OpenWeather API

## How to Run
1. Clone the repo
2. Run:
   ./mvnw spring-boot:run
3. Open:
   http://localhost:8080
